# MLX Steer

Activation steering and cognitive monitoring for local LLMs on Apple Silicon.

> Shape your local model's behavior at inference time — no fine-tuning needed.

**Coming soon.** Star this repo to get notified.

## What this will do

- Compute steering vectors from contrastive prompt pairs
- Inject vectors into transformer hidden states during generation
- Monitor cognitive health (entropy, repetition, goal drift) in real-time
- Works with any MLX-compatible model (Gemma 4, Qwen 2.5, Llama 3)

Built on [MLX](https://github.com/ml-explore/mlx) by Apple.
